# Wing

简洁流畅、数据驱动，响应式的WordPress主题。

![Wing](https://biji-cdn.wuzii.com/wp-content/uploads/图像2022-5-3-20.36-scaled.jpg)

## 重要！

主题**无jQuery**，与很多插件可能有各种各样地兼容问题，如遇到兼容性问题请尝试关闭PJAX。
